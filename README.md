# MSCS_634_ProjectDeliverable_2

## Regression Modeling and Performance Evaluation

---

## Project Overview
This project focuses on developing and evaluating regression models to predict a target variable using machine learning techniques. The primary objective is to improve predictive accuracy through feature engineering, model selection, and performance evaluation.

---

## Dataset Description
The dataset used in this project contains multiple predictor variables (features) and one target variable.

Data preprocessing steps included:
- Handling missing values using mean imputation
- Converting categorical variables into numerical format using one-hot encoding
- Ensuring data consistency and cleanliness before modeling

---

## Feature Engineering
Feature engineering was applied to enhance model performance and extract meaningful patterns from the data. The following techniques were used:
- Creation of ratio-based features
- Log transformation to normalize skewed data
- Removal of less important features

---

## Models Implemented
- Linear Regression
- Ridge Regression
- Lasso Regression

---

## Model Evaluation Metrics
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R-squared (R²)

---

## Cross-Validation
5-fold cross-validation was applied to ensure model generalization and prevent overfitting.

---

## Results Summary
- Ridge Regression performed best overall
- Lasso Regression helped with feature selection
- Linear Regression showed slightly higher error

---

## Key Insights
- Feature engineering improved model performance
- Regularization reduced overfitting
- Cross-validation confirmed model stability

---

## Challenges
- Missing data handling
- Feature selection
- Model tuning

---

## Solutions
- Used data imputation
- Applied feature transformations
- Compared multiple models
